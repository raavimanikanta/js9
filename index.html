Objects in javaScript

Objects are non-primitive datatypes.In objects the data is stored as key and value pairs .we can store any dataTypes in object.

Example:
         const obj={
             name:"manikanta",
             id:20,
             college:"pragati",
         }

Different ways to create the objects

Objects can be created in three ways:

  • object literals
  • object instance
  • function constructor

object instance

In this type objects are created by using the new keyword.

Example:
         const object=new Object() 
             object.name="manikanta"
             object.id=22,
             object.college="pragati"
         console.log(object)
         Output:
         object:-
         {
            college: "pragati"
            id: 22
            name1: "manikanta"
         }

Object literals

Objects are in the normal manner and it is the simple of creating of objects.

Example:
         const obj={
             name:"manikanta",
            age:23,
            location:"mumbai"
         }

Objects using function constructor

These are used to create the objects in js .Objects are created using new keyword .This method is useful for creating the objects with similar properties.

Example:
         function Person (name,age,college) {
             this.name=name;
             this.age=age;
             this.college=college;
         }
         const newObj=new Person ("Varma" ,23,"Narayana")
         console.log(newObj)

Objects Keys and objects value

Object Keys

Object.keys are used in js,it returns all key propeties of the object.

Example:
         var obj={
             name:"manikanta",
             age:23,
             college:"pragati"
         }
         console.log(Object.keys(obj))

    output:['name', 'age', 'college']

Object value

Object.values are used in js,it returns all values of the object.

Example:
         var obj={
             name:"manikanta",
             age:23,
             college:"pragati"
         }
         console.log(Object.values(obj))

    output:['manikanta',23, 'pragati']

Spread operator

Spread operator is used to separate the elements of array ,string ,or objects to the individual elements.

Example:
        const objSpread={
            name:'manikanta',
            age:23,
        }
         const objSpread2={
             ...objSpread,
             college:"Pragati"
         }
         console.log(objSpread2)
     output:
     objSpread2={
         name:"manikanta",
         age:23,
         college:"pragati",
     }
